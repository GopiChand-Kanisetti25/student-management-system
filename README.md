# Student Management System
A console-based Student Management System developed using Python.

## Features
- Add student
- View students
- Search student
- Update student
- Delete student
- File-based storage

## Technologies Used
- Python
- Jupyter Notebook

## How to Run
1. Open the notebook in Jupyter
2. Run all cells
3. Follow menu instructions
